    grayscale

    @function grayscale($gradation, $alpha) { ... }

    Description

    The grayscale function takes two arguments:

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $gradation

    There are 10 levels of gray, ranging from 1 to 10 (darkest to lightest).

    		Number—none
    $alpha

    There are 11 levels of opacity, ranging from 0.0 to 1.0 (transparent to opaque).

    		Number—none

    Example

    grayscale(1, 1);
This will create a dark gray that is fully opaque.

    Throws

    • Only alpha values in 0.1 increments are allowed

    Requires

    • [variable] DBgrayscaleColors
    • [variable] __alpha-stops

    color

    @function color($name, $properties...) { ... }

    Description

    Color helper used a color by it's name, select a shade and modify opacity

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $name

    Color named defined in $DBglobalColors. There are 7 colors (gray, blue, green, red, yellow, black, white).

    		String—none
    $properties...

    Each color has 6 shades (x-dark, dark, base, medium, light, x-light). Each shade has 11 levels of opacity, ranging from 0.0 to 1.0 (transparent to opaque).

    		String or Number—none

    Example

    This will create a base blue that is fully opaque.

    color(blue);

    This will create a dark red that is 80% opaque.

    color(red, dark, 0.8);

    This will create a base yellow that is 60% opaque.

    color(yellow, 0.6);

    Throws

    • Only alpha values in 0.1 increments are allowed

    Requires

    • [variable] __shade-vals
    • [variable] __alpha-stops
    • [variable] DBglobalColors

    Used by

    • [mixin] nest-check
    • [mixin] list-ui
